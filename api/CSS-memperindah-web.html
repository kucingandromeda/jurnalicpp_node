CSS,sang memperindah web

perkenalan dengan CSS

Desain yang kalian lihat di setiap website yang memperindah dan memperkeren tampilan website itu bukanlah perkerjaan HTML. Semua itu dilakukan oleh CSS(Cascading Style Sheet) yang mengatur semua tampilan visual yang ada di dalam website.

Alasan CSS dibuatpun simpel dikarenakan Dulu HTML tidak dilengkapi dengan tags yang berfungsi untuk memformat halaman.Conthnya tag seperti <font></font> yang diperkenalkan pada HTML versi 3.2 meyebabkan masalah dalam pengembangannya yang membuat code panjang dan susah di baca. Oleh sebab itu, W3C(World Wide Web Consortium) membuat CSS untuk menyelesaikan masalah ini.

Hingga kini CSS dan HTML memiliki hubungan yang sangat erat dan saling berkerja sama dalam pengembangan website, HTML yang berfungsi sebagai bahasa markup (fondasi situs) dan CSS memperbaiki style (untuk semua aspek yang terkait dengan tampilan website), maka kedua bahasa pemrograman ini harus berjalan beriringan.

cara kerja CSS

CSS seperti bahasa inggris sederhana yang berbasis syntax yang diatur dengan rule.Jadi bagaimana cara jika ingin memberikan visual pada code ini?

<p>halo saya seorang manusia</p>

Cara kerja CSS itu sederhana, struktur terdiri dari selector dan declaration block.Pilih element lalu declarasi yang akan dilakukan kepada element tersebut.Jika contoh codenya seperti ini:

<style> p { text-align: center; font-size: 16px; color: pink; } <style>

Jadi mudahkan dalam menggunakan CSS.Sekian dulu perkenalan singkatnya dengan CSS dan tentu saja ada lebih banyak lagi hal yang menarik dalam CSS ini, namun sampai disini saja.
